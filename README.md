# Style GAN in PyTorch

PyTorch implementation of [Style GAN](https://arxiv.org/abs/1812.04948).

## Dependencies

- Python 3.6.8
- PyTorch 1.3

## Dataset

[FFHQ](https://github.com/NVlabs/ffhq-dataset)
